# YAML

- Syntax:
key: value
- Indentation: space(s), creates objects
- Comments: '#'
- Data types:
  - Int
  - Float
  - Strings: with or without quotes
  - Null: ~ or null
  - Bollean: True or On | False or Off
  - Arrays:
    list: [a, b, c] or
    list:
      - a
      - b
      - c
  - Dictionaries
    object: {a: 1, b: 2, c: 3} or
    object:
      a: 1
      b: 2
      c: 3
